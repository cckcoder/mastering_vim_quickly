## The "dot" command
example: You want to append `;` to all three lines.
you could run a command like
`A;<ESC>j`

```
my $i
my $learn
my $quickly
```

then press `. (dot)` to repeat the change in next line

```
my $i;
my $learn;
my $quickly;
```

ref: `:help change.txt`
